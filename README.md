# umi project
# Getting Started

# 使用集团账户中心模块，需要切换集团haier源
  npm i nrm -g  
  nrm add haier http://npm.haier.net/  
  nrm use haier  

# 安装依赖
  依赖集团源haier，只能用npm安装
  npm install

# 开发调试
  npm run Start

# 构建
  npm run build
